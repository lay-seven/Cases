一、继承性
特性：子元素有默认继承父元素样式的特点
可以继承的常见属性（文字控制属性都可以继承）
不是控制文字的都不能继承

1.color
2.font-style font-weight font-size font-family
3.text-indent text-align
4.line-height
5...br 6.继承失效的特殊情况：
(1)a标签的color会继承失效
超链接颜色属性继承失效
(2)h系列标签的font-size会继承失效

h系列字体大小继承失效

二、层叠性
1.给同一个标签设置不同的样式，此时样式会层层叠加，共同作用在标签上
2.设置相同的样式，此时样式会层层覆盖，最终写在最后的样式会生效
3.当样式冲突时，只有当选择器优先级相同，才能通过层叠性哦啊不断结果
